Predicting Students Adaptivity Level in Online Education

Katie Jiang - AI Camp DS Crash Course

Let's Start

What does the project do?

In the recent years, the outbreak of Covid-19 changes peopleâ€™s life. Lots of classes move from in-person to remote. Students' learning environment and equipment for joining online school varies from person to person. The teaching mode not only changes students learning speed, but their attitudes towards school work. Thus, how do those changes affect students' digestive of course work? In this project, we aim to identify the important features that affect the learning quality and make predictions for student's digestive based on those factors.

The dataset we will use contains information about student's learning environment such as gender, age, device type, internet quality and so on. All of the features are categorical. Feature importance will be used to help students do self-evaluation when he/she is taking online courses.

Exploration

Data Sources and Structure

The dataset we used is obtained from Kaggle called "students_adaptability_level_online_education" and can be found here: Student Adaptability Level Data. The dataset contains 14 columns and 1205 instances, each corresponding to a studnet. It is not surprise that all of the column variables are categorical, becasue this data is quite like a questionnaire.

The dataset has one response variable, Adaptivity Level, which contains three levels: high, moderate and low. After plotting the distribution of the response varible, we find it is imbalanced.

We can plot the number of each class for our target variables to show that it is a highly imbalanced dataset:

Feature exploration: Gender Effects

From the plot of adaptivity level vs. gender, we coud observe that males tend to digest new knowledge faster compared to females and the level of low adaptativity level is almost the same for both genders.
We plotted the following bar chart to show the distribution of each sex:

Feature exploration: Age Effects

Different age group tends to have various learning abilities, from the age group, we could visualize that the optimal age range of learning for students is from 11-15 and 21-25, among them, 21-15 is the range that more students reach high adaptivity level compared with other age range.

Feature exploration: Class duration Effects

The reasonable class duration is also an important factor for student to digest course work efficiently. From the plot, we could see that class duration is best to be 1-3 hours for students to capture knowledges. If the class time beyond this limit, student may feel boring and hard to focus.


Also, we visualized one interesting trend of financial condition. People in middle class has the most number of adapting materials, which in turns, digesting more knowledges comparing to families with high and low financial condition.

Data Preparation

To deal with the imbalanced dataset, we used the following two techniques:
1. Applied stratified splitting when splitting the dataset into train, validation and test sets.
2. Processed the data using Synthetic Minority Oversampling Technique (SMOTE).
To convert the categorical variables into numeric data, we apply the label encoding techniques to map the categories into 1, 0 coding(LabelEncode ). This approach is very simple and it involves converting each value in a column to a number.

Models

Our main goal is to classify the student digestive level based on the learning environment. Since the tree and boosting methods perform very well in general. We decide to start with the baseline model: Logistic Regression and move from there.
In total, we will use the Logistic regression, Support Vector classifier, Random Forest and Xgboost as our candidate models. After training process, we will evaluate performance of each models based on confusion matrix and F1 score. Also we are going to identify the important features that contribute a lot in predicting adaptivity level.

Logistic Regression

  • Logistic regression is a common predictive model used in lots of classification problem. Its structure makes it easy to deploy and interpret. It has an assumption that all samples are independent, identically distributed and the conditional distribution of the target given a sample is Bernoulli perfectly fits our data.
  • We tune two hyper-parameters in logistic regression - maximum number of iterations and the solver.
  • We used lbfgs solver for learning a classifier. The result of logistic regression model has an average accuracy of 60% for three classes.
  • By considering the feature importance, the model suggests financial condition, self-limitation and institution types were identified as the most important features

    • We plot the confusion matrix to visualize how our model is predicting the test data. Also, we make graphs on the feature importance of features to see whether our models make sense intuitively.

    Random Forest

  • Random forests is a supervised Machine Learning Algorithm that is used widely in Classification and Regression problems. It builds decision trees on different samples and takes their majority vote for classification and average in case of regression. It is very easy to build and train. When we walk through the categorical variables, the tree based model could learn each split, which is similiar with the answers in questionnaire.
  • The training process is pretty same as the logistic regression models, we performed 5-fold cross validation to find the best setting for two hyperparameters - the maximum depth and the number of trees/estimators.
  • The result of the model is really good as we expected. The overall accuracy for each class is about 93%, the precisions for all class are above 90% and the f1 score for all class are also above 90%. For the important features, the random forest gives slightly different top 3 features: financial conditions, class duration and age are detected as important features by random forests model.

    • XGBoost

  • Extreme Gradient Boosting is a type of gradient based boosting that is known to have good performance and speed. It belongs to a family of boosting algorithms and uses the gradient boosting (GBM) framework at its core. It is an optimized distributed gradient boosting library. XGBoost implements several optimization and regularization techniques and handles sparse data very well. This model is also expected to fit the data well.
  • Three hyperparameters are tuned: the number of trees, maximum depth and learning rate. We tune them by grid search using 5-fold cross validation.
  • XGBoost models performs quite steady and has close result as random forest. The overall average of accuracy is 93%, the precisions for all class are above 90% and the f1 score for all class are also above 90%. The little difference performance between the Xgboost and the random forest is Xgboost perform better in class 3, whearas random forest tends to perform better in class 2.
  • The top 3 important features captured by this model are class duration, financial conditions and the self-limitation.

    • Support Vector Classifier

  • Support vector classifier is to find a hyperplane in an N-dimensional space(N â€” the number of features) that distinctly classifies the data points. For nonlinear case, SVC could map the data into higher dimension using different kernel function. In our case, we convert our categories into numeric and it will be easy to fit the SVC.
  • We tuned the C(penalty term), gamma, degree and kernel types using a grid search and 5-fold cross validation. Initially, the model does not perform well, but after tuning the reasonable parameters, the accuracy increases by 0.3.
  • The result of our tuned model has 87% accuracy on average, with above 85% precision score for all classes. However, in recall, it suggest the model performs bad in predicting class 3 and extremely well in predicting class 1. Becasue the nonlinear kernel does not give the feature importance, we extract the feature importance from linear kernel and show the institution type, financial condition and class duration are top 3 features.

    • Model Comparison

    In this section we will focus on comparing models based on confusion matrix. Becasue confusion matrix is a really good evaluation metric in classification.

    We have plotted the percentges of corrected predicted in each class for each model. There are four heatmaps corresponding to four models' performance.

    Since the data is highly imbalanced, it is important to care more about the accuracy with in each class rather than the overall accuracy. Thus, we could say the random forest and the xgboost models are the optimal ones in this classification problem, given the fact that they all have 90% accurate prediction in each class.

    Conclusions

  • We trained 4 different models to make predictions for student adaptivity level. The model selections are based on the data structures and the types of features.
  • We see that Random forests and the Xgboost are the best performing model and all models achieve the 80% accuracy score in classifying the target variable. Also, we could notice that finanical condition and class duration are very importance features no matter which model we pick. It is interesting to see that gender and internet type are actually not that significant in attending online education. In other words, as long as the class setting is reasonable and family supports finanical cost, everyone is expected to digest course work well.
  • To further imporve the project, we could modify our model to serve as a self-assessment in online learning. Asking user to input their learning background such as age, gender, class duration and internet types etc., then our system will output a general studying quality based his/her background. If we want to be more precise, we will collect more data such as colleges provided online education in geographically, the class structures, and the popularity of the teacher among students. For the model selection, we will stick on some tree-based and boosting methods, becasue they perform really well in categrical data structures.
